# Task: Create DrawChatOverlay Component

## Requirements
Create `src/components/DrawChatOverlay.tsx` with:
- Collapsible semi-transparent panel overlaying right side of whiteboard
- Shows last ~20 messages from the #draw channel
- Collapsed by default with toggle button to expand
- Semi-transparent background (bg-black/60)
- Compact message format: username + text only
- Simple text input at bottom for sending messages
- Positioned absolutely within whiteboard container
